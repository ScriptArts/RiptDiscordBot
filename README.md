# RiptDiscordBot
DiscordBotのりぷとちゃん

## コマンド

コマンドエイリアス: `!`

|コマンド                     | 説明                              |
|:---------------------------|:-----------------------------------|
| `!help`                     | 使えるコマンドの一覧を表示します   |
| `!ping <arg1>`                     | 指定したアドレスにPingを送信します |
| `!chat <arg1>`              | りぷとちゃんと会話します             |
